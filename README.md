# Vendetta Fonts

A collection of fonts for Vendetta and forks.

## Available fonts

- [Noto Sans](./NotoSans.json)
  - [Condensed](./NotoSansCondensed.json)
  - [Semi Condensed](./NotoSansSemiCondensed.json)
  - [Extra Condensed](./NotoSansExtraCondensed.json)
- [Inter](./Inter.json)
- [Geist](./Geist.json)

## How to use

1. Copy the link to the font you want to use.
2. In Vendetta or a fork, go to **Settings** > **Vendetta** or **fork name** > **Fonts**.
3. Select the **Plus** button.
4. On the **Import Font** page, select **Import font entries from link**.
5. Paste the link into **Font Link**.
6. Select **Import**.
7. In the font list, select **Apply** next to the font you added.
